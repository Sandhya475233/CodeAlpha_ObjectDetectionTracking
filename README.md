# AI Object Detection and Tracking

This project performs real-time object detection and tracking using **YOLOv8** and **OpenCV**.  
The model detects objects from video input and displays bounding boxes with object labels.

## Features

- Real-time object detection
- Object tracking using YOLOv8
- Video processing using OpenCV
- Displays detected objects with bounding boxes
- Saves processed output video

## Technologies Used

- Python
- OpenCV
- Ultralytics YOLOv8
